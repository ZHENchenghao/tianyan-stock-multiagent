# 天眼 v9 · 完整推理链

## 总览

```
输入层 (layer1/layer2/layer3)
  │
  ├─ 9维独立评分 (每个维度产出 score∈[0,100] + direction)
  │
  ├─ 新闻间接修正 (news_delta → 渗透 宏观/资金流/景气度)
  │
  ├─ S_total 合成: z-score映射 → 权重归一化 → Φ(z)加权求和
  │
  ├─ 后验概率: P_bull / P_neutral / P_bear + 信息熵
  │
  ├─ 迟滞环: 双模态状态机 → 防抖
  │
  ├─ 刚性截断: DEFENSE/PANIC/CRISIS → 仓位强制归零
  │
  └─ 裁决: 加仓/持有/观望/减仓 + 纠错线
```

```

  PANIC           → score=20, neutral  (准确率0%, 错杀反弹)
  CRISIS          → score=30, neutral  (准确率0%, 错杀反弹)
  DEFENSE_SHOCK   → score=30, bearish  (准确率51%)
  DEFENSE_*       → score=35, bearish
  CAUTION_*       → score=50, neutral  (准确率34%, 太偏空)
  NORMAL          → score=70, neutral

v9新闻修正: news_delta × 0.6 → 渗透宏观分
  例: news=-10 → 宏观 35→29 (-6分)

## 三、S_total 合成 (verdict_math)

### 步骤1: z-score映射
```
z_i = (score_i - 50.0) / 25.0
→ score=0  → z=-2.0
→ score=50 → z= 0.0
→ score=100→ z=+2.0
```

### 步骤2: 权重归一化
```
raw_weights = {宏观:0.25, 大盘:0.20, 景气:0.15, 趋势:0.15,
               资金流:0.15, 盈亏:0.10, 压力:0.10, 反共识:0.10, 规则:0.05}
Σraw = 1.25
w_norm_i = raw_i / 1.25  → Σw_norm = 1.0
```

### 步骤3: S_total计算
```
S_total = Σ (w_norm_i × (2Φ(z_i) - 1))
其中 Φ(z) = 0.5 × (1 + erf(z/√2))  ← math.erf实现, 零外部依赖

S_total ∈ [-1.0, +1.0], 天然收敛, 不会溢出
```

### 步骤4: 后验概率
```
输入: z_vals + S_total
输出: P_bull, P_neutral, P_bear, σ_ensemble, entropy, confidence_band

信息熵越低 → 信号越一致
置信带宽越窄 → 估计越精确
```

### 步骤5: 迟滞环
```
双模态状态机, 防频繁切换:
  P_bear > enter_threshold → 进入看空状态
  P_bull > exit_threshold   → 退出看空状态
  dead_zone: 中性区不产生信号
```

## 五、裁决输出

```
无刚性截断时:
  S_total > 0.25        → 加仓
  0.10 < S_total ≤ 0.25 → 持有偏多
  -0.10 ≤ S_total ≤ 0.10→ 观望/持有
  -0.25 ≤ S_total < -0.10→ 观望偏空

  → 静态观望 (无论S_total多大)
  → 纠错线: 等待宏观体制解除
